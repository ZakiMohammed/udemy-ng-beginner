# Notesy App - Angular Getting Started

Angular notes app (notsey) getting started.

## Run Application
```
npm start
```

## Initial Setup

```
ng new ng-1-init
```

## Add Dependencies

```
npm i bootstrap bootstrap-icons bootswatch
```

## Add Components

```
ng g c header
ng g c footer
ng g c empty
ng g c form
ng g c item
ng g c list
```
